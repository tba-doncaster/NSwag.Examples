# NSwag.Examples (Metapackage)

  **⚠️ This is an empty metapackage for backwards compatibility only.**

New projects should use:

- **[TBA.NSwag.Examples.AspNetCore](https://www.nuget.org/packages/TBA.NSwag.Examples.AspNetCore)** - For ASP.NET Core
- **[TBA.NSwag.Examples.Core](https://www.nuget.org/packages/TBA.NSwag.Examples.Core)** - For .NET Framework Web API

See [the GitHub repository](https://github.com/tba-doncaster/NSwag.Examples) for documentation and examples.
