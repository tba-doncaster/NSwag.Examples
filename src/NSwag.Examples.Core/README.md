# NSwag.Examples.Core

Core library for TBA Doncasters's maintained fork of `NSwag.Examples`. This package contains the annotations and base logic for defining NSwag examples as code, including `ReflectionExampleRegistry` for use in non-dependency-injected projects.

See [the github repository](https://github.com/tba-doncaster/NSwag.Examples) for more information & examples.
